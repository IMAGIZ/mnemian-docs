# `digital_object`

**Type**: Object

---

The `digital_object` field in AtoM contains metadata about a single digital representation (Instantiation) attached to the description. AtoM supports one "master" digital object per description, typically a medium resolution image or PDF, plus automatically generated reference and thumbnail versions.

This field provides an opportunity to showcase the full potential of RiC-O modeling with comprehensive digital object representation.

### RiC-O Entities

1. `rico:Instantiation` (the digital object instantiation)
2. `rico:CarrierType` (categorizes the digital carrier)
3. `rico:RepresentationType` (categorizes the content representation)
4. `rico:InstantiationExtent` (file size and dimensions)
5. `rico:ExtentType` (what is being measured)
6. `rico:UnitOfMeasurement` (bytes, pixels, etc.)
7. `rico:Identifier` (URLs for different versions)
8. `rico:IdentifierType` (URL type categorization)
9. `rico:Date` (upload timestamp)

### Example

```turtle
# Core record resource
:RecordSet_123 a rico:RecordSet ;
  rico:title "Administrative Records, 1950-1960"@en ;
  rico:hasOrHadDigitalInstantiation :DigitalObject_123 .

# Digital object instantiation
:DigitalObject_123 a rico:Instantiation ;
  rico:hasCarrierType :CarrierType_Digital ;
  rico:hasRepresentationType :RepresentationType_Image ;
  rico:hasExtent :FileSize_123 ;
  rico:hasOrHadIdentifier :MasterURL_123, :ReferenceURL_123, :ThumbnailURL_123 ;
  rico:hasOrHadDate :UploadDate_123 ;
  rico:note "Filename: sassy-bunny.jpg"@en ;
  rico:note "MIME type: image/jpeg"@en .

# Carrier type
:CarrierType_Digital a rico:CarrierType ;
  rdfs:label "Digital file"@en .

# Representation type
:RepresentationType_Image a rico:RepresentationType ;
  rdfs:label "Image"@en .

# File size extent
:FileSize_123 a rico:InstantiationExtent ;
  rico:hasExtentType :ExtentType_FileSize ;
  rico:hasUnitOfMeasurement :UnitOfMeasurement_Bytes ;
  rico:value "29458"^^xsd:integer .

# Extent type
:ExtentType_FileSize a rico:ExtentType ;
  rdfs:label "File size"@en .

# Unit of measurement
:UnitOfMeasurement_Bytes a rico:UnitOfMeasurement ;
  rdfs:label "bytes"@en .

# URL identifiers
:MasterURL_123 a rico:Identifier ;
  rico:textualValue "http://www.example.com/uploads/r/artefactual-archives/f/1/d/f1d57e030432177172e638fab2b995fb191b9861cdd9f7b424e3f3c759be0aa9/sassy-bunny.jpg"@en ;
  rico:hasIdentifierType :IdentifierType_MasterURL .

:ReferenceURL_123 a rico:Identifier ;
  rico:textualValue "http://www.example.com/uploads/r/artefactual-archives/f/1/d/f1d57e030432177172e638fab2b995fb191b9861cdd9f7b424e3f3c759be0aa9/sassy-bunny_141.jpg"@en ;
  rico:hasIdentifierType :IdentifierType_ReferenceURL .

:ThumbnailURL_123 a rico:Identifier ;
  rico:textualValue "http://www.example.com/uploads/r/artefactual-archives/f/1/d/f1d57e030432177172e638fab2b995fb191b9861cdd9f7b424e3f3c759be0aa9/sassy-bunny_142.jpg"@en ;
  rico:hasIdentifierType :IdentifierType_ThumbnailURL .

# Identifier types
:IdentifierType_MasterURL a rico:IdentifierType ;
  rdfs:label "Master URL"@en .

:IdentifierType_ReferenceURL a rico:IdentifierType ;
  rdfs:label "Reference URL"@en .

:IdentifierType_ThumbnailURL a rico:IdentifierType ;
  rdfs:label "Thumbnail URL"@en .

# Upload date
:UploadDate_123 a rico:Date ;
  rico:value "2016-01-01T17:00:00"^^xsd:dateTime ;
  rico:hasDateType :DateType_Upload .
```
