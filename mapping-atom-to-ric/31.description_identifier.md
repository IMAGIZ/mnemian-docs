# `description_identifier`

**Type**: String

---

The `rico:Identifier` entity is a word, number, letter, symbol, or any combination of these used to uniquely identify or reference an individual entity instance. It connects the following pieces of information:
- **The resource being identified**: The `rico:RecordSet` (`Record`/`RecordPart`) or `rico:Person`, `rico:CorporateBody` (etc.) entity that the identifier belongs to, which links to the identifier using `rico:hasOrHadIdentifier`.
- **The actual text of the identifier**: In the `rico:textualValue` object property, attached directly to the `rico:Identifier` entity.
- **Temporal information**: The time period during which this identifier was valid by using `rico:Date` and properties like `rico:wasUsedFromDate`, `rico:wasUsedToDate`.
- **Type of identifier**: Relationships to a `rico:IdentifierType` entity. For this field, we use a shared `rico:IdentifierType` whose `rdfs:label` is "Description identifier"@en.

### How this differs from the main archival Identifier or Reference Code 

The main archival identifier (often called "Reference code") may be physically affixed to a real‑world item and is modeled separately with an `IdentifierType` labeled "Reference code". The AtoM `description_identifier` is a separate identifier for the description itself (sometimes mirroring the reference code, sometimes different). We keep the model simple by creating a distinct `rico:Identifier` for the description value and typing it with the shared `IdentifierType` labeled "Description identifier".

### RiC-O Entities
1. `rico:Identifier` (description identifier, textual value from AtoM)
2. `rico:IdentifierType` (shared, reused; labeled "Description identifier")
3. `rico:RecordSet/Record/RecordPart` (resource that has the identifier)

### Example

```turtle
# Define the IdentifierType (shared) entity and its label
:IdentifierType_Description  a  rico:IdentifierType ;
    rdfs:label "Description identifier"@en .

# Example record set entity
:RecordSet_123  a  rico:RecordSet ;
    rico:hasOrHadIdentifier  :DescriptionIdentifier_XYZ .

# Create the description identifier entity
:DescriptionIdentifier_XYZ  a  rico:Identifier ;
    rico:textualValue "DESC-123-ABC"@en ;
    rico:hasIdentifierType  :IdentifierType_Description .
```
